# Samantha-Smith-s-Portfolio-website
Professional marketing portfolio website with editable projects + instructor evaluation

## Project images

- Add project screenshots/files to [images/projects](images/projects).
- Add supporting files for new projects (PDFs/slides/docs) to [images/projects/assets](images/projects/assets).
- Use relative paths in project fields (for example: `images/projects/grammy-campaign.jpg`).
- For assets, use paths like `images/projects/assets/your-project-file.pdf` in **Media Link**.
- You can also use full hosted URLs if your media is online.
- Suggested naming pattern: `project-name-01.jpg`, `project-name-02.jpg`.
- For multi-part series, use project numbers: `charity-water-project-1-...`, `charity-water-project-2-...`, `charity-water-project-3-...`.
- Examples from current projects: `grammy-awards-campaign-01.jpg`, `charity-water-landing-page-01.jpg`, `intel-csr-metrics-01.jpg`.

### Quick image checklist

- Preferred formats: JPG or PNG.
- Recommended size: under 2MB per image.
- Recommended dimensions: 1600x900 (16:9) for consistent presentation.
- Keep file names lowercase and hyphen-separated.
- Store files in [images/projects](images/projects).
- Store non-image files in [images/projects/assets](images/projects/assets).
- Use paths like `images/projects/your-file-name.jpg` in project links/media fields.
